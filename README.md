### Kotlin Maven Kick Start

#### Getting started

- clone projet
```shell
$ git clone https://github.com/telkomdev/KotlinKickStart.git
```

- build and download dependencies
```shell
$ mvn clean package
```

- Run 
```shell
$ java -jar target/KotlinKickStart-1.0-SNAPSHOT-jar-with-dependencies.jar
```

- Unit test
```shell
$ mvn test
```
